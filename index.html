What happens when you include import java.util.Scanner; in your code? Where does this functionality come from? What is the java in java.util.Scanner;?

The statement java.util.Scanner; allows you to accept input from the termianal or different sources. It basically means to import Scanner class which is in a util folder, inside the java folder. The use of it so far, has been to accept information from the keyboard into your program. There are other methods of input with this utility other than the keyboard, but I have not put any of them to the test at this stage.

To generate a random number you use Math.random();? What is Math? What is the return type of the .random() method. Why is the M in Math capitalized?

Math. is capitalized in java because its a class. The .random is the function - you can have other extensions after Math such as Math.sqrt( this would give you the square root of the value). Basically, all Math.random does is generate a random double with a positive sign in the range of greater than 0.0 and less than 1.0. Because these numbers are rarely of use we can add or subtract to them to give us a specific range of randomly generated numbers to be used.
